# PSO_routes
基于粒子群的cvrp问题建模求解
对一系列装卸货点进行适当的路径规划，
在满足约束条件（客户需求、车辆载重和容积、车型、车辆行驶里程、配送时间窗、配送中心数量等限制）
和目标最优化（路程最短、成本最低、使用车辆数最少、配送时间最快等）下，
将客户的配送需求从配送中心送达客户点，或从客户点送回配送中心。
